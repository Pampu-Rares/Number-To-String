
# Number-To-String

A static webpage which lets you enter any positive number and see its complete string.

![The webpage in question](./Icons/site-preview.png)

You can try this webpage for yourself [here](https://pampu-rares.github.io/Number-To-String/)


## Getting Started

### Installation

1. Simply paste this into your terminal:

```shell
git  clone  https://github.com/Pampu-Rares/Number-To-String.git
```

2. You can now host this static page locally

That's it.
